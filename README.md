# UPI Transaction Analysis Dashboard (Power BI)

## Problem Statement

This project focuses on analyzing UPI transaction data to understand transaction patterns, balance trends, and user behavior.  
The dashboard helps in tracking transaction activity over time, monitoring remaining balance after each transaction, and identifying spending patterns across different cities and currencies.  

It also enables comparison of transaction activity across months, helping in identifying trends and usage behavior.

---

## Steps Followed

- Step 1 : Loaded dataset into Power BI Desktop  
- Step 2 : Performed data cleaning and transformation using Power Query Editor  
- Step 3 : Designed interactive dashboard using charts and slicers  

- Step 4 : Created visuals:
  - Multi-line charts for transaction amount trends  
  - Balance tracking after each transaction  
  - Matrix visualization for detailed transaction analysis  
  - Month-wise comparison of transactions  

- Step 5 : Added slicers and synchronized them across pages  

- Step 6 : Implemented Bookmark and Selection Pane for dynamic view switching  

- Step 7 : Designed dual-page report with interactive navigation
